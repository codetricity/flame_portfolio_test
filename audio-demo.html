Flame Audio Demo

The demonstration on the left may take 30 seconds to load the first time. It is built with Flutter Web and designed for mobile devices. I did not optimize the Flutter mobile app for the web. This is an example to show that Flutter Web does work with flame_audio.

After the app loads, it will take several seconds for the audio file to load from GitHub. If you are patient, you will be able to hear the audio files eventually.

Code for the project is available https://github.com/codetricity/flame_audio_tutorial
